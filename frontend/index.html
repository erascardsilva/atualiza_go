Sobre

Atualiza GO — Atualizador de Sistema Linux

Como Usar

  1. Na tela Início, confira se o sistema foi detectado corretamente.
  2. Acesse Atualizar e selecione os pacotes que deseja atualizar (Sistema, Flatpak, Snap).
  3. Clique em Iniciar Atualização. O sistema pedirá sua senha via autenticação gráfica.
  4. Acompanhe o progresso pelas barras e pelo log de saída em tempo real.
  5. Após concluir, use a tela Ações para reiniciar ou desligar, se necessário.
